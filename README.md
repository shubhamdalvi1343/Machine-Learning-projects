K-Means Clustering on Customer Spending Data
📌 Project Overview

This project implements K-Means Clustering to segment customers based on their spending behavior. The goal is to identify distinct customer groups that can help businesses make data-driven marketing and business decisions.

The model uses the Elbow Method to determine the optimal number of clusters and applies data preprocessing techniques to improve clustering performance.

📂 Dataset

The dataset used in this project:

File Name: Cust_Spend_Data.csv

Contains customer-related features such as spending patterns and other numerical attributes.

The first two columns are excluded during clustering (as per notebook logic).
